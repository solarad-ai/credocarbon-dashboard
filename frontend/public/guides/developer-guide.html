Creating a Project

Step Description
1. Project Type Select category: Renewable Energy, Energy Efficiency, Forestry/AFOLU, Waste Management, or Industrial Processes
2. Basic Information Enter project name, description, location, start date, and crediting period
3. Generation Data Upload emission reduction data, specify monitoring methodology, add baseline calculations
4. Stakeholder Consultation Upload consultation records, meeting minutes, and community engagement documentation
5. Compliance Complete social and environmental safeguards checklists
6. Registry Submission Select target registry (GCC, VCS, Gold Standard) and submit for validation
